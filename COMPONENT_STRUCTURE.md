# Dharti Solar Website Structure

This document outlines the planned component and section structure for the premium solar business website built with Astro + Tailwind CSS.

## Sections & Components

1. Sticky Navbar (components/Navbar.astro)
2. Hero Section (components/Hero.astro)
3. Services Section (components/Services.astro)
4. Why Choose Us (components/WhyChooseUs.astro)
5. Subsidy Section (components/Subsidy.astro)
6. Projects Section (components/Projects.astro)
7. CTA Section (components/CTA.astro)
8. Contact Section (components/Contact.astro)
9. Footer (components/Footer.astro)
10. Floating WhatsApp Button (components/WhatsAppButton.astro)

Each section will be implemented as a separate Astro component for clean structure and reusability.
